# E-commerce-Sales-Dashboard

📊 Power BI Project | E-commerce Sales Dashboard

I have developed an interactive E-commerce Sales Dashboard using Power BI, focusing on deriving meaningful business insights one of my recent project.
Key Highlights:
 -  Analyzed Year-to-date (YTD) vs previous-year-to-date (PYTD) sales to track performance trends
 -  Calculated Year-over-Year (YoY) growth for better comparison
 -  Built category-level insights (Furniture, Office Supplies, Technology)
 -  Designed KPI indicators (▲▼) to highlight performance changes
 -  Created dynamic and user-friendly visuals for easy decision-making
   
Key Learnings:
 -  Data cleaning and transformation using Power Query
 -  Creating measures using DAX
 -  Designing interactive dashboards
 -  Understanding business insights from data
   
Business Insights:
 -  Identified top-performing and underperforming categories.
 -  Analyzed sales trends across different time periods.
 -  Highlighted growth opportunities using data-driven insights.

This project enhanced my understanding of data visualization and business insights.
